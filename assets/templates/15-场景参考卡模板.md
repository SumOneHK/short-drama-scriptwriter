## 资产 ID 命名格式（全 skill 统一规范）

格式：`SCN_{LocationType}_{Place}_{Mood}_{v01}`

- 全大写下划线分隔，避免空格 / 中文 / 特殊字符
- `LocationType`：场景大类，如 `INT`（室内）/ `EXT`（室外）/ `MIX`（半开放）
- `Place`：地点英文短名，如 `Mansion / Office / Hospital / School / Alley`；同名地点用 `_A / _B` 区分
- `Mood`：场景情绪标签，如 `Day / Night / Rainy / Wedding / Funeral / Confrontation`；可省略
- `v01`：版本号（场景升级 / 翻新时递增）

示例：
- `SCN_INT_Mansion_LivingRoom_Wedding_v01`
- `SCN_EXT_RoofTop_NightStorm_v02`
- `SCN_INT_Office_CEORoom_Confrontation_v01`

避免：
- 不要写中文 ID（如 `继母客厅`）；中文写在 `场景中文名` 字段
- 不要在 ID 里塞集号（如 `_EP005`）；集号写在 `首次出现集数` 字段
- 不要用空格 / 句号 / 斜杠分隔
