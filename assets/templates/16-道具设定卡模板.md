## 资产 ID 命名格式（全 skill 统一规范）

格式：`PROP_{Category}_{Name}_{State}_{v01}`

- 全大写下划线分隔
- `Category`：道具大类，如 `KEY`（钥匙类核心物）/ `DOC`（文件 / 信件 / 合同）/ `JEWEL`（首饰）/ `WEAPON` / `TECH`（手机 / 电脑）/ `SYMBOL`（象征物）
- `Name`：道具英文短名，如 `MedicalRecord / WeddingRing / NDA / FamilyPhoto / OldKey`
- `State`：可选，状态变体，如 `Original / Faded / Burnt / Sealed / Bloodied`；用来区分同一道具的不同状态
- `v01`：版本号

示例：
- `PROP_DOC_MedicalRecord_Original_v01`
- `PROP_JEWEL_HeritageRing_v01`
- `PROP_KEY_OldMansionKey_Rusted_v02`

避免：
- 不要在 ID 里塞中文；中文写 `道具中文名`
- 不要塞集号；集号写 `首次出现集数 / 状态变化` 字段
