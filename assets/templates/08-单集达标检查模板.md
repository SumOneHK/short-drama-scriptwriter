- 情节承接是否成立：
- 冲突升级是否成立：
- 价值闭环是否成立：
- 下集勾连是否成立：

## 五、开场与承接

- 上一集尾钩核心短语：
- 第一场承接落点：
- 开场第一动作或第一句对白：
- 开场是否直接带出上一集尾钩中的人物 / 道具 / 地点 / 动作：

## 六、正文厚度

- 是否存在“简卡层厚、正文层薄”：
- 是否有场次只剩 `1-2` 条动作 + `1-2` 轮对白就直达结果：
- 场次正文动作/反应/对白行数是否明显多于前置字段：
- 常规集是否达到目标行数：`45-75秒 >=22行 / 60-90秒 >=28行 / 90-120秒 >=34行`：
- 首集是否达到目标行数：`75-120秒 >=34行 / 90-150秒 >=40行`：
- 最薄的一场需要补的第二动作：
- 最薄的一场需要补的第二阻力：
- 最薄的一场需要补的第二反应：
- 每场是否有可见后果：

## 七、节奏与密度

- 情绪落点是否形成变化：
- 本集中段错误放松或小释放：
- 本集新关键信息点：
- 信息投放是否都挤在结尾说明：
- 是否存在“设定正确但读起来费劲”：
- 抽象规则是否已翻译成可见后果：
- 屏幕字是否像规则说明书：
- 是否只抛新债、不推进旧债：
- 是否重复上一集的回报或钩子方式：
- 正文是否撑得住目标时长：
- 是否存在靠解释对白硬拉时长：

## 八、对白与画面

- 是否存在自报家门式台词：
- 是否存在解释前情、重复已知信息的台词：
- 是否存在工作稿语言进入正文：
- 是否存在跨题材套话或角色声线不贴当前场景：
- 对白是否发生在动作里：
- 本集最值钱画面：
- 关键场是否有视觉记忆点和动作升级点：

## 九、对白口语化与本地化

通用项（所有项目）：
- 单句长度是否默认 `<= 15` 字（中文）或 `<= 12` 个 word（英文）：
- 是否存在书面化、剧本腔、新闻腔台词需要改成口语：
- 是否存在解说词式台词替观众做总结：
- 同场对手两人句长 / 句式 / 词汇锐度是否能听出差别：
- 把对白单独提出来去掉 `角色名：` 后，是否仍能凭句式判断是谁在说：
- 这一集是否至少有一句"只属于这个角色 / 这段关系 / 这个场景"的台词：
- 删掉"最说明的那一句"后这场是否仍然成立；如果成立是否已经删：

国内项目专项（命中时填）：
- 是否仍有老旧剧本腔（`你这个混蛋 / 你以为你是谁 / 我会让你付出代价`）：
- 是否仍有翻译腔或书面化（`难道你不知道吗 / 我无法接受这种事实`）：
- 称呼系统是否符合关系层级（关系近用 `哥/姐/老张`，翻脸场用全名，不要全场全名互称）：
- 网感词是否只放在配角 / 龙套 / 群像，主角声线是否独立成立：
- 网感词是否使用 `半年以上稳定词`，没有出现三个月内的过气热梗：
- 方言使用是否轻量受控，非方言区观众是否能秒懂：
- 是否给关键对峙场留了至少一次"压住一句话不说"的瞬间：

海外项目专项（命中时填）：
- 是否所有对白都是英文 + 中文注释，没有 `英文角色名：中文对白` 混写：
- 中文注释是否在补 `潜台词 / 情绪意图`，而不是字面翻译；是否默认 `<= 15` 字：
- 英文对白是否完全 contractions 化（`I'm / don't / won't / gonna / wanna`）；如果出现 `do not / I am / cannot` 是否有刻意提级理由：
- 是否仍存在翻译腔（`How dare you / You are courting death / I will never forgive you`）需要替换成自然口语：
- 角色 register 是否分层正确（teen / young adult / professional / antagonist / parental）：
- cliffhanger 句式是否避开了同批次重复模式：
- 感叹 / 咒骂强度是否符合本项目分级（PG / dark romance / thriller / 成人）：
- 是否出现需要本地化的中国元素（`face / save face` 类）但没有彻底改写或显式标注：
- 是否出现欧美语境不可接受 trope（家暴复合 / 抓手腕强吻 / 包办婚姻命令式）需要按 `dark romance` 重新设计：

## 十、格式规范

- 场次标题是否统一为 `### 1-1 日内 地点`：
- 每场是否都有 `人物：` 行：
- 首次出场人物是否在 `人物：` 行补身份说明：
- 动作行是否统一使用 `△` 或约定 `A：`：
- 台词是否统一使用 `角色名：`：
- 广播 / 录音 / 系统播报是否用 `VO：`：
- 内心独白 / 未出镜短句是否用 `OS：`：
- 手机 / 闸机 / 弹窗 / 文件 / 名单 / 弹幕等画内信息文字是否用 `屏幕字（类别）：`，并与 `屏幕字（叠加）：` 区分：
- 关键声音是否用 `SFX：`：
- 是否存在 `广播：/ 机械音：/ 系统提示：/ 黑伞灯浮字：` 等未归一标记：
