## 三点五、改编合规风险预判

改编项目天然带版权与敏感线，必须在立项前显式说清：

- 版权状态：`已授权` / `公版` / `未授权（需走授权流程）` / `不确定`
- 真实人物风险：原著是否含真实历史人物或在世公众人物；是否需要改名 / 化用 / 隐去
- 真实事件风险：原著是否影射真实社会事件、真实案件、真实组织；改编后是否仍可识别
- 原著敏感线：原著本身是否含未成年涉性 / 极端暴力 / 民族宗教 / 政治隐喻 等敏感内容
- 平台兼容性：原著名是否在目标平台有过下架 / 限流 / 投诉记录
- 改编尺度边界：哪些情节必须保留（原著粉敏感），哪些必须删改（合规要求）
