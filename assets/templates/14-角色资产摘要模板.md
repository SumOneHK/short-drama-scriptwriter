## 资产 ID 命名格式（全 skill 统一规范）

格式：`CHR_{Tier}_{NameEN}_{LookVariant}_{v01}`

- 全大写下划线分隔
- `Tier`：角色层级，如 `LEAD`（主角 / 核心关系位）/ `ANTAG`（核心对手）/ `SUPP`（主要配角）/ `FUNC`（功能角色）
- `NameEN`：角色英文名（不含空格），如 `AvaWilson / CarolineSnow / LuciusBlack`
- `LookVariant`：可选，造型变体，如 `Default / Wedding / Hospital / Disguised`
- `v01`：版本号

示例：
- `CHR_LEAD_AvaWilson_Default_v01`
- `CHR_ANTAG_CarolineSnow_Funeral_v01`
- `CHR_SUPP_DanielReed_Lawyer_v02`

避免：
- 中文角色名写 `中文名` 字段，不进 ID
- 不在 ID 里塞集号
