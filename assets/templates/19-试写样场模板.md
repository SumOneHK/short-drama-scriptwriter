# 试写样场

## 使用规则

- 只在以下情况启用：
  - 主角声线还不稳
  - 核心关系张力还不稳
  - 主角首次亮相还不够立
  - 关键对峙场还不够活
  - 高概念设定第一次落地还拿不准
- 优先试写 `1-3` 个样场，不要扩成完整分集正文
- 样场不替代 `分集大纲` 和正式剧本
- 样场如果不成立，优先回改 `人物小传 / 故事大纲 / 故事节拍表 / 分集大纲`
- 这份文件只解决“这一场能不能立住”，不记录长期连续性、不安排整批推进

## 自检

- 主角声音是否立住：
- 关系张力是否成立：
- 动作和场面调度是否成立：
- 是否至少留下一个视觉记忆点：
- 是否出现明确动作升级或权力变化：
- 本场结束后，局势或情绪场是否真的变化：
- 是否已经形成可见后果，而不是只多说了一层解释：
- 是否需要回改上游文件：

### 对白口语化与本地化复检

- 单句长度是否满足 `<= 15` 字（中文）或 `<= 12` word（英文）：
- 把对白单独提出去掉 `角色名：` 后，是否仍能凭句式判断是谁在说：
- 是否至少有一句"只属于这个角色 / 这段关系 / 这个场景"的台词：
- 国内项目：是否仍有书面化、剧本腔、翻译腔；称呼系统是否符合关系层级：
- 海外项目：是否完全 contractions 化（`I'm / don't / won't / gonna`），中文注释是否补潜台词不是字面翻译：
