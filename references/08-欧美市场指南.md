# TikTok 欧美市场指南

当目标平台是 TikTok 的欧美英语市场时，使用本指南。

## 基本原则

这是“本地化原创开发”。海外项目要从美国英语市场的关系逻辑、职业语境、校园语境和 trope 系统出发重新组织。

如果用户没有指定地区，默认按当代美式英语处理。

## 平台特征

- 卖点必须一眼看懂；高概念是可选策略，不是所有项目的前提。
- 如果选择高概念，规则必须能用普通观众一句话复述，不能靠长解释撑住。
- 行为逻辑、恋爱逻辑、校园或职场逻辑要符合欧美语境。
- 对话必须像真人会说的话，不能像翻译腔。
- 情绪冲突可以强，但不要无脑堆砌中式狗血。

## 常见有效引擎

除非用户明确另有要求，否则优先考虑：
- Dark romance：痴迷、背叛、禁忌吸引、敌对恋爱
- Supernatural romance：狼人、女巫、吸血鬼、诅咒血脉
- YA / campus thriller：人气、排挤、社团、身份崩塌
- Family secret thriller：继承、失踪、隐瞒身世、掩盖真相
- Reinvention arc：逃离有毒环境、重塑身份、风格化复仇
- Survival / sci-fi：实验失控、身体互换、异界入口、淘汰机制

美国英语市场优先参考：
- [16-美国英语市场选题与设定词库.md](16-美国英语市场选题与设定词库.md)

使用提醒：
- 复用同一套框架：`目标受众 / 题材类型组合 / 核心设定组合 / 风格元素组合`
- 海外项目优先使用美国英语市场词库，再按项目做细化
- 如果项目来自国内母体，先把它改写成美国语境里天然成立的关系、权力和欲望结构

## 对白要求

底层逻辑：英文必须是美国当代用户在 TikTok 短剧里"会一口气说出来"的口语，不是中式英语直译，也不是英文教材腔。

完整方法（contractions 硬规则 / 翻译腔反例库 9 条 / register 5 档分层 / cliffhanger 句式池 / 感叹分级 / 中文注释格式）见 [12-单集写作协议.md 第七章 7.3 节](12-单集写作协议.md)，本节不重复展开。

海外项目快速自检（写完每场过一遍）：
- 海外线只让 `对白 / 旁白 / 屏幕字` 使用英文，策划层和动作行 `△` 仍是中文
- 不出现 `英文角色名：中文对白` 混写
- 每句英文对白后紧跟中文注释，格式 `English line.（中文注释。）`，注释补潜台词不是字面翻译，默认 `<= 15` 字
- 所有英文对白完全 contractions 化（`I'm / don't / won't / gonna / wanna`）；未刻意提级时不出现 `do not / I am / cannot / will not`
- 没有翻译腔（`How dare you / You are courting death / What is the meaning of this`）
- register 分层稳定（teen / young adult / professional / antagonist / parental）
- cliffhanger 句式同批次不重复
- 感叹 / 咒骂强度符合本项目分级（PG / dark romance / thriller / 成人）

## 本地化要求

底层逻辑：海外项目不是"国内剧本翻成英文"，而是"用美国当代社会纹理重新组织故事"。完整文化锚点池见 [16-美国英语市场选题与设定词库.md](16-美国英语市场选题与设定词库.md) 的 `四点五、文化锚点与美国当代社会纹理`，国内母体重写矩阵见 [02-策划细则.md](02-策划细则.md) 的 `国内母体 → 美国市场 trope 重写矩阵`。

基础规则：
- 角色、主要地点、关键道具统一成 `中文 / English` 形式
- 职业、学校、住房、交通、社交参照物要符合当地常识
- 家庭压力、社会压力、恋爱压力和权力关系要优先使用美国语境天然成立的版本
- 策划层和动作描述仍用中文，方便内部协作

## 题材 → 机构落地（美国语境）

不同题材必须用美国原生机构和原生压迫机制，不要套国内模板：

- 网红 / creator → algorithm shadow ban / brand sponsor pulled / cancellation campaign / lawsuit threat / NDA
- 学校 → school district / detention / suspension / college rescinded / Common App / SAT / scholarship
- 法律 → restraining order / class action / NDA / non-compete / wrongful termination / Miranda
- 财务 → trust fund / probate / 401k / vesting / IRS audit / bankruptcy / FICO
- 医疗 → ER bill / out-of-network / medical bankruptcy / 5150 hold / inpatient / hospice
- 婚姻 → prenup / postnup / alimony / custody battle / restraining order / divorce decree
- 移民 → green card / H-1B / DACA / ICE / asylum / mixed-status family
- 职场 → HR complaint / hostile work environment / PIP / RIF / wrongful termination / union strike

## 美国主流价值观弧光

正面（默认可用）：
- `American Dream` / `Underdog` / `Self-made` / `Grit / resilience` / `Comeback / redemption arc` / `Reinvention` / `Found family` / `Speaking truth to power` / `Owning your truth`

价值观红线（默认避免 — 这些在美国 = 红旗，不是爽点）：
- `Toxic / toxic relationship`：精神控制；只能写给反派
- `Gaslighting`：让对方怀疑自己的认知
- `Love bombing`：开局过度示好再撤回作为操控
- `Red flags`：早期警示信号（控制欲、嫉妒、隔离）
- `Abuse / DV / IPV`：家暴；不能美化也不能用"爱"包装
- `Stalking`：跟踪 / 监视；主角追求要避开
- `Coercive control`：经济控制 / 信息控制 / 隔离

特别提醒：
- "霸总" 在美国语境里要么变成 `dark romance`（明确风格化、禁忌、双方知情），要么变成 `protective / loyal CEO`（保护欲不是控制欲）
- "公开打耳光 / 当众羞辱" 要小心；美国版本是 `viral takedown / receipts dropped / public callout / social media destruction`，靠舆论而不是肢体
- "抢婚 / 强娶 / 包办婚姻" 不能直接套；要改成 `last-minute objection at the wedding`（合法 trope）或 `arranged marriage with consent`

## 节日 / 季节锚点（天然剧情触发器）

把关键集卡在节日上，比抽象 cliffhanger 更有锚定感（详见词库 `节日 / 季节锚点`）：

- Thanksgiving 家庭秘密在火鸡桌上揭露（11 月）
- NYE midnight 接吻 / 不接吻（12 月底）
- Halloween 派对身份揭穿（10 月）
- Prom / homecoming queen 公开对峙（春季）
- Fourth of July 烟花夜约会 / 决裂（7 月）
- Christmas 家庭团聚揭露伪装（12 月）

## 多元身份与移民线

如果项目涉及多元身份，优先按真实文化层处理：
- First-gen / 移民二代：`code-switching` / `bilingual at home` / `cultural community pressure` / `arranged-vs-love marriage` / `family honor`
- 涉及 LGBTQ+ 主线：可以写恋爱主线，但避免把性向冲突写成"治愈" / "拯救"
- 跨族裔 romance：要写出真实文化纹理（家庭反应、语言切换、节日冲突），不要 `colorblind casting`

## 政治 / 宗教敏感线（默认回避，除非剧情核心）

- 党派政治（Democrat / Republican / Trump / Biden）：默认避免明示
- 枪支管控、堕胎权、移民政策：高度极化，默认不站队
- 具体宗教派别仪式：写到具体仪式时要文化顾问
- 可以写信仰冲突，但不要写宗教对立

文化锚点池（让台词长在美国语境里，不是抽象 globalized 英语）：
- 学制：`senior year / junior / freshman / prom / homecoming / SAT / ACT / GPA / valedictorian / detention / principal's office`
- 法律 / 财务：`restraining order / NDA / prenup / 401k / credit score / SSN / IRS / lawyer up / Miranda rights`
- 节日 / 季节：`Thanksgiving / Halloween / Black Friday / Memorial Day / Fourth of July / spring break`
- 居住 / 地理：`zip code / HOA / suburbs / uptown / the Hamptons / brownstone / trailer park`
- 餐饮 / 日常：`Starbucks run / drive-thru / takeout / leftovers / Uber / DoorDash`
- 校园 / 社交：`group chat / DM / posted it / went viral / cancel / ghosted / blocked`

文化禁区（直接套用国内桥段会翻车）：
- 男主"霸道占有"、抓手腕、强吻、语言贬损 → 美国语境里属于 abuse / harassment，需要重新设计为 `dark romance` 框架并显式信号化
- 家暴 + 复合 / 出轨 + 被原谅：在欧美语境里属于不可接受 trope，必须改成 `离开 + 报复 + 重塑` 的弧光
- 家长包办婚姻、长辈强行干预成年子女恋爱：必须改成 `cultural community pressure` 而不是直接命令式
- 公开打耳光、当众羞辱不是默认爽点；要先建立 stakes 再放
- 不要把中国元素直接英译塞进对白（如 `face / give face / save face`）；要么彻底本地化，要么显式标注 cultural setup

## 节奏要求

- 画面推进要快，少解释，多发现、多威胁、多选择。
- 结尾钩子优先用揭露、危险、背叛、吸引、不可逆决定。
- 前 `3` 集的钩子优先直接落到现场后果，尤其是公开场、权力场、恋爱对峙场；不要只停在“文件异常 / 有人可疑 / 某条记录不对”。
- 保留一些潜台词，不要把每个感受都用台词讲穿。

## 商业化与付费卡点

海外短剧主要分两种付费形态：

- TikTok 原生 / 切片导流：免费 + 广告；本身不挂付费卡点，但首屏 `0-15s` Hook（"explosion point"）的强度直接决定从 TikTok 到独立 App 的转化
- 海外独立短剧 App / 付费系列：典型 IAP 付费阶梯——免费章 `EP001-EP010` 拉钩，首付费卡 `EP010-EP015` 必须放 `S` 档强翻面，`EP015` 之后按解锁阶梯每 `3-5` 集一次 `A` 档以上钩子持续付费

写作要求：
- 命中独立 App 付费形态时，`分集大纲.md` 必须按 `S / A / B` 标注卡点强度
- 首付费卡的强翻面优先选：身份揭露、关系背叛、权力反转、生死危机、禁忌触发
- 海外项目的 cliffhanger 不要复用国内"打脸 + 反咬"的固定句式，要落到欧美语境天然成立的 trope 上

## 合规重点

海外项目（TikTok / 美国 App Store / Google Play）高频踩坑：
- 性内容：未成年涉性绝对红线；成人内容必须按平台分级
- 暴力 / 血腥：过度图像暴力 / 自残 / 自杀图像在 TikTok 会限流或下架
- 仇恨言论：种族 / 性别 / 性取向 / 宗教 / 残障的歧视性描写
- 涉枪 / 涉毒 / 涉酒：美国 App Store 有明确分级要求，TikTok 对儿童可见内容有额外限制
- 真实人物 / 公众人物：未授权使用名人姓名 / 形象，可能触发 right of publicity 诉讼
- 文化挪用 / 刻板印象：少数族裔、原住民文化符号误用
- 心理健康议题：抑郁 / 焦虑 / 自残的不当呈现可能触发平台干预

立项与策划阶段动作：
- `项目简报.md` 的 `合规风险预判` 字段必须按 TikTok 内容政策 + Apple App Store / Google Play 分级要求预判
- 涉枪 / 涉毒 / 涉自残必须显式标注降级策略；未成年人涉性零容忍
- 不做"中式狗血直接英文翻译"——很多国内常见情节（如家暴 + 复合、男方占有欲极强的"霸总"行为）在欧美语境里属于不可接受 trope
