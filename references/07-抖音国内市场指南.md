# 抖音国内市场指南

当目标平台是中国大陆抖音短剧或漫剧时，使用本指南。

## 平台特征

- 开头 `1-3` 秒必须给钩子。
- 关系、身份差、压迫感要立刻看懂。
- 情绪要明确，不要故意模糊。
- 每集都要有明显回报、打脸、反转或新危机。
- 适配竖屏和碎片化观看。

## 常见有效引擎

除非用户明确要小众题材，否则优先从这些引擎里选：
- 身份反转：隐藏身份、废柴变强、真假千金、失落继承人
- 关系拉扯：先婚后爱、契约关系、强强对抗、破镜重圆
- 家庭与阶层：豪门、职场、亲情压力、利益联姻
- 玄幻奇观：都市异能、古风玄幻、系统、妖灵、风水、修罗场
- 爽感复仇：打脸虐渣、夺回资源、事业翻盘、自我觉醒

立项提醒：
- 国内漫剧不默认要求高概念。主流稳态题材只要爽点、压迫和回报足够清楚，也可以成立。
- 如果做创新，优先做 `类型微创新`：换压迫机构、关系结构、公开翻面场或视觉符号，而不是一上来堆复杂规则。

## 写法要求

- 台词短、狠、明确。
- 重要信息尽量用冲突抛出来，不要靠解释。
- 现代题材优先使用中国观众熟悉的社会关系和生活结构。
- 一场戏只围绕一个主要冲突点推进。

## 对白口语化（国内项目硬要求）

底层逻辑：观众听到的是"这个人在这种情境下真会说出来的话"，不是"剧本里的人念稿"。

完整方法（反例库 / 称呼系统 / 声线分层 / 网感词与方言边界 / 潜台词省略）见 [12-单集写作协议.md 第七章 7.2 节](12-单集写作协议.md)，本节不重复展开。

国内项目快速自检（写完每场过一遍）：
- 单句 `<= 15` 字，超过就拆
- 没有老旧剧本腔（`你这个混蛋 / 你以为你是谁 / 我会让你付出代价`）
- 没有翻译腔或书面化（`难道你不知道吗 / 我无法接受这种事实`）
- 称呼符合关系层级（关系近用 `哥/姐/老张`，翻脸场才全名）
- 网感词使用 `半年以上稳定词`，不撑主角
- 同场两个对手把对白单独提出去掉 `角色名：`，仍能凭句式判断是谁在说
- 关键对峙场至少留一次"压住一句话不说"的瞬间

## 本地化要求

底层逻辑：题材 / 人物 / 世界观 / 矛盾 / 爽点都要长在中国当代真实社会纹理里，否则观众会秒出戏。完整文化锚点池见 [15-国内选题与设定词库.md](15-国内选题与设定词库.md) 的 `四点五、文化锚点与当代社会纹理`。

基础规则：
- 人名、身份、家庭结构、行业逻辑要符合中国语境
- 当代题材要使用中国观众能自然接受的机构和社会规则
- 不要出现"英文壳中文芯"的假本地化行为
- 当代场景锚点要落地：单位、户口、社保、年终、KPI、租房中介、外卖、滴滴、相亲角、家长群、业主群、菜市场、菜鸟驿站、共享单车——这些是观众秒懂的现实纹理，比抽象设定更有效
- 不要把欧美 trope 直接翻成中文：`alpha / beta / 命定伴侣 / fated mates / prom / 毕业舞会` 在国内当代语境里默认违和，要么彻底本地化（高考志愿 / 毕业典礼 / 谢师宴），要么明确标注为校园奇幻 / 海外背景

## 题材 → 机构落地（避免万剧一面）

不同题材必须用原生机构和原生道具，不要所有题材都写成"保安 / 登记册 / 门禁"：

- 直播 / 网红 → 切流 / 限流 / 封号 / 热搜 / 中之人 / MCN / 平台审核 / 投诉率
- 校园 → 储物柜 / 缺勤记录 / 班主任 / 教导处 / 奖学金 / 社团 / 同伴排挤 / 家长群
- 末日 → 污染 / 物资票 / 安全区资格 / 配给 / 临时秩序
- 体制 → 处分 / 党纪 / 述职 / 巡视 / 调离 / 待岗
- 医疗 → 病危通知书 / 自费签字 / 医闹 / 飞刀 / 多学科会诊
- 法律 → 调解书 / 执行 / 老赖 / 失信名单 / 取保
- 商战 → 股权代持 / 对赌协议 / 上市 / 突击入股
- 互联网大厂 → 优化 / 调岗 / PIP / N+1 / 竞业 / 期权回购
- 豪门家族 → 老宅产权 / 信托 / 家族基金 / 婚前财产 / 私生子 / 祠堂

完整对应表见 [15-国内选题与设定词库.md](15-国内选题与设定词库.md) 的 `题材-机构落地对应`。

## 国内主流价值观爽点

主流爽点（默认可用）：
- `寒门贵子` / `扮猪吃老虎` / `装弱反杀` / `以德报德、以直报怨` / `家国情怀` / `公开清算` / `投射打工人` / `豪门下嫁 + 反向追妻`

价值观红线（默认避免，需要时显式标注）：
- `绿茶 / 白莲花`：只给反面角色，不给主角
- `PUA`：只能写给反派或主角早期被害再反杀
- `捞女 / 凤凰男`：地域 + 性别污名标签，谨慎使用
- `躺平`：剧本里的"躺平"通常是假装放弃再翻
- `富即正义 / 拜金`：要包装成 `事业 + 自尊`，不能直接拜金

## 地域分层与剧情纹理绑定

不同地域有天然剧情纹理，选题时优先匹配（详见词库 `地域分层与城市绑定`）：

- 一线（北上广深）：北漂 / 沪漂 / 大厂 / 学区房 / 高端医美
- 新一线（杭州 / 成都 / 武汉 / 西安 / 重庆）：互联网新贵 / 文创 / 慢生活
- 二三线 / 县城：人情社会 / 国企家庭 / 体制内 / 小镇做题家
- 东北 / 广东 / 川渝 / 江浙沪：各自天然底色

提醒：
- 选定地域后，对白、场景、配角都要顺这条纹理走
- 不要一边主角说北方话一边场景写成上海弄堂

## 海外母体改国内的重写要求

`美国母体 → 国内市场 trope 重写矩阵` 已统一在 [02-策划细则.md](02-策划细则.md) 同一处与 `国内母体 → 美国市场` 矩阵并列维护，避免双份维护漂移。

国内项目处理海外母体（如 `dark romance / werewolf alpha / Ivy League scandal / billionaire CEO romance`）时：
- 不直译，先按 02-策划细则 的 `美国母体 → 国内市场 trope 重写矩阵` 逐条重写
- 重写完成后，在 `锁题摘要.md` 显式写出"母体来源市场 + 至少 3 条最大重写动作"
- 填不出 3 条说明本地化深度还不够，回到本指南的"题材底层逻辑 / 价值观爽点 / 视觉符号"再做一次现场重写

## 节奏要求

- `60-90` 秒一集，通常 `3-5` 场。
- 最小结构是：钩子、压力、转折、结尾钩子。
- 同一种羞辱或打脸不能机械重复，必须升级代价或范围。

## 商业化与卡点（后置）

剧本阶段默认先判断故事是否好看、是否能连载、是否符合抖音竖屏观看语法。用户没有明确商业化模式时，统一记录为 `商业化后置`，不按 IAA / IAP / 平台付费定制倒推题材、大纲或单集字段。

国内项目明确商业化后，常见形态主要有三种（卡点设计完全不同，详见 [24-卡点设计.md](24-卡点设计.md)）：

- `IAA`：抖音端原生 / 信息流 / 短剧 App 免费段；卡点核心是完播、留存、中插广告卡位和追更钩；不走付费阶梯
- `IAP`：小程序付费短剧；典型阶梯是免费章拉钩、首付费卡强翻面、后续按解锁阶梯持续付费
- `平台付费定制 / 会员独享`：长视频平台定制，不走小程序卡点，但首尾留更长情绪兑现

商业化明确后的写作要求：
- IAA 项目：每集按 `5s 开场 / 25s 中段刷新 / 50s 二次升压` 三个完播抓手设计；连续 2 集追更钩类型重复时按 `需修改` 处理；中插广告卡位选场不能在情绪高点
- IAP 项目：`分集大纲.md` 按 `S（强翻面）/ A（强钩子）/ B（普通钩子）` 标注卡点强度；免费章、首付费卡集、解锁阶梯由项目商业方案决定，不在未确认商业化时预设
- 平台付费定制：长头部建制（首集 / 前 3 集），中后段每 5-8 集一次大事件
- 商业化未明确时，不填上述商业卡点字段，只保留剧作层的 `本集观众一句话 / 本集情绪回报 / 本集尾钩 / 尾钩如何改变下一集目标、危险、关系或信息差`

## 合规重点

国内短剧高频踩坑题材：
- 涉未成年：校园暴力、未成年人恋爱、霸凌中的过度残忍描写
- 涉公职：警察 / 检察官 / 法官 / 军人形象不严肃化或被反转
- 涉医：虚假疗效、医患极端冲突、超纲医疗操作
- 涉法：私刑正义、教唆犯罪、把违法行为正当化
- 涉民族 / 宗教：少数民族文化符号误用、宗教仪式戏剧化
- 价值观红线：拜金、炫富、家暴美化、PUA 关系美化、复仇至上
- 历史人物 / 真实事件：未授权使用真名、明显歪曲史实

立项与策划阶段动作：
- `项目简报.md` 的 `合规风险预判` 字段必须写清当前题材最可能踩到的 `1-3` 条红线
- 长线 / 平台定制项目走北京市广播电视局或对应平台的立项备案绿色通道
- 改编项目按 `04-故事理解模板.md` 的 `改编合规风险预判` 走一遍，再进策划
