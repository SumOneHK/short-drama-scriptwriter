# 卡点设计

`卡点` 有两层含义：
- 剧作层：让观众继续看的具体戏剧问题、状态改变和尾钩
- 商业层：为 IAA / IAP / 平台付费服务的广告位、付费点或会员段设计

本文只在用户明确要求商业化、投流、广告、付费解锁，或需要专项诊断追更钩时读取。常规剧本阶段先看故事质量，不用 IAA / IAP 倒推选题和大纲。

## 〇、默认口径：商业化后置

商业化未定时：
- 不默认 IAA
- 不强制填写广告卡位、激励视频、付费触发位、S/A/B 档
- 分集大纲只需要写清剧作层的 `本集观众一句话 / 本集情绪回报 / 本集追更钩 / 尾钩如何改变下一集`
- 选题和剧本质检优先看人物、冲突、情绪、类型承诺和 TikTok 竖屏可读性

用户明确商业化后，才进入下表：

## 底层逻辑：商业卡点服务于商业化模式

**不同商业化模式，卡点目标完全不同**。明确商业化模式后，再设计商业卡点；不要在用户未要求商业化时用商业卡点限制剧本。

| 商业化模式 | 关键指标 | 卡点目标 | 卡点形态 | 主流场景 |
| --- | --- | --- | --- | --- |
| `IAA`（广告变现） | 完播率 / 留存 / CTR / VCR | 让用户看完本集 + 立刻刷下集 + 不在中插流失 | `完播抓手 / 追更钩 / 广告卡位 / 激励视频卡` | 抖音 / 快手 / TikTok 端原生；ReelShort 等 episode-app 免费部分 |
| `IAP`（付费解锁） | 付费率 / ARPU / 阶梯转化 | 让用户掏钱解锁后续 | `S/A/B 档付费卡` | 国内小程序付费短剧；ReelShort 付费部分 |
| `平台付费定制` | 会员留存 / 完播 / 付费续订 | 留住会员 + 集数完播 | `长头部建制 / 会员独享 / 中后段稳定密度` | 长视频平台定制（爱奇艺 / 优酷迷雾剧场风格） |

**默认假设已取消**：当前 skill 不再把未明确商业化的项目默认按 `IAA` 处理。本文仍保留 IAA / IAP / 平台付费定制方法，供明确商业化时调用。

判断口径：
- `06-需求清单.md` 的 `商业化模式` 字段决定是否进入本文件的商业卡点设计
- 不写 `商业化模式` 或写 `商业化后置` 时，不追加商业字段
- 同一项目若同时分发到多渠道（如抖音 IAA + 小程序 IAP），必须写两套卡点版本，或在某一版基础上做最小改造

## 一、IAA 模式卡点设计（主流）

IAA 模式下"卡点"指的是**让用户不流失、看完、刷下集**的所有戏剧设计。每集要同时承担三种卡点：完播卡 + 追更卡 + 广告卡位。

### 1.1 完播卡：让用户看完这一集

`60-90` 秒单集流失高峰在前 5 秒、第 25 秒（约 1/3 处）和第 50 秒（约 2/3 处）。每集必须设计 3 个抓手分别对应这三个流失高峰：

`抓手一：前 5 秒"开场抓力"`
- 必须落在第 1-2 个动作行；不允许从背景介绍 / 角色独白 / 解释设定开场
- 推荐形态：可见冲击（公开打耳光 / 名册空白 / 同僚视而不见）/ 反差登场（最低身份说出最高判语）/ 危机爆发（生死瞬间 / 关系撕裂瞬间）
- 反例：主角在车里独白、女主在镜子前思考人生、角色介绍式画外音

`抓手二：第 25 秒"中段刷新"`
- 不能让前段建立的局势平稳延续；必须在 1/3 处给一次小翻面或新信息
- 推荐形态：第二阻力（对手加压）/ 关系翻面（朋友变敌人）/ 新人物入场 / 道具揭露 / 受害者出现
- 反例：从第一场到中段都是同一组人物对话同一件事

`抓手三：第 50 秒"二次升压"`
- 在结尾钩子之前必须再升一次压；不能让中段以后直接平推到结尾
- 推荐形态：反派反制 / 主角误伤 / 真相部分揭露 / 公众反应介入 / 时间压力增加
- 反例：中段以后直接进入"主角胜利"或"主角失败"，没有第三次振荡

### 1.2 追更卡：让用户立刻刷下集

IAA 模式下"追更"是行为触发（用户主动刷下集）而非付费触发。追更卡 = 让用户产生"下一集会发生什么我必须知道"的强烈具体好奇。

追更卡的硬要求（每集结尾必须满足）：
- `具体性`：必须能让观众用一句具体话复述他想知道什么；不允许"暗示有秘密 / 留下伏笔"等抽象词
  - 算成立：`柳照雪命牌背后还有一个被盖住的字 / 男主递的名片上是律所名 / 母亲手里那张照片里有第三个人`
  - 不算成立：`下一层秘密出现 / 真相即将揭晓 / 故事将进入新阶段`
- `时效性`：观众必须感觉"答案下集就能拿到"，而不是"早晚会知道"
- `行动锚点`：钩子里必须出现具体可行动元素（人物 / 地点 / 道具 / 时间），让下集第一场可以直接承接
- `状态改变`：钩子必须改变下集的目标 / 危险 / 关系 / 信息差中的至少一项；只增加未解之谜不算

追更钩类型库（同批次相邻集不重复同型）：
- `身份揭露`：你知道她是谁吗 / 你和我妈是什么关系
- `背叛`：你做的 / 一直都是你
- `权力反转`：你以后听我的 / 我才是真正的继承人
- `禁忌触发`：你敢出去 / 再说一句试试
- `生死危机`：他还活着 / 她不见了 / 后面有人
- `选择困境`：你只能救一个 / 现在就决定
- `时间压力`：明天之前 / 24 小时

### 1.3 广告卡位：让中插广告不杀完播率

中插广告在剧情高潮 / 情绪顶点切入会直接劝退用户。广告卡位的设计原则是"让广告插在状态切换的余震里"：

可插广告的位置（按平台节奏）：
- 单集 60 秒以内：原则上不插中插广告，只用前贴 / 后贴
- 单集 60-90 秒：可插 1 次中插，建议放在第 35-40 秒（中段反应余震处）
- 单集 90 秒+：可插 1-2 次中插，建议放在 30 秒 / 65 秒附近

广告卡位选场原则：
- `选`：动作完成后的反应回合 / 状态变化的余震 / 一场结束到下一场的过渡
- `不选`：紧张对峙的中段 / 接吻或冲撞的瞬间 / 主角即将做关键选择的瞬间 / 真相揭露的爆点
- 写法暗示：在写正文时就预留"余震 1-2 行"作为隐性广告位，标注 `△（情绪余震，可作中插点）`，便于后期插入

### 1.4 激励视频卡（IVR）

激励视频是 IAA 模式下"用户主动看广告换内容"的卡点机制。适合长线项目设计"剧外激励"：

适用场景：
- 解锁本集番外片段
- 解锁角色内心独白 / 写真级镜头
- 解锁下一集预告片完整版
- 解锁结局选择（多结局项目）

设计原则：
- 激励视频解锁的内容必须是"看了爽、不看不影响主线理解"
- 不允许把主线关键情节藏在激励视频后；那是 IAP 卡点逻辑，不是 IAA
- 激励内容长度建议 15-30 秒；激励视频本身 15 秒，性价比合理
- 激励触发位选在"主角刚拿到关键情绪点 / 关键关系建立瞬间 / 关键道具初登场后"

### 1.5 IAA 模式的单集结构模板（默认 60-90 秒）

```
0-5s    [完播卡-开场抓力]   可见冲击
5-25s   [建制段]            交代本集起点 + 第一动作 + 第一阻力
25s     [完播卡-中段刷新]   小翻面 / 新阻力 / 新人物
25-50s  [升压段]            主动作推进 + 反应链 + 第二阻力
~35s    [广告卡位]          情绪余震处可插中插（如适用）
50s     [完播卡-二次升压]   反派反制 / 真相部分揭露 / 时间压力
50-60s  [收束段]            动作翻面 + 状态变化
60s+    [追更卡-尾钩]       具体钩 + 时效性 + 行动锚点
```

写每集前，按这个模板把每个时间段对应的具体动作填出来；填不出来的回改大纲。

## 二、IAP 模式卡点设计（小程序付费短剧 / 海外 episode-app 付费部分）

IAP 模式下"卡点"指的是**让用户掏钱**的关键集结尾设计。前面 IAA 的完播/追更/广告卡仍然适用，IAP 在此之上额外增加付费卡机制。

### 2.1 付费阶梯典型设计（国内小程序剧）

```
EP001-EP005   全免费，强钩子拉留存
EP006-EP010   全免费，密度逐渐拉满
EP010-EP015   首付费卡集（必须 S 档强翻面，不可逆）
EP015 之后    每 3-5 集放一次 A 档以上钩子，按解锁阶梯递进
EP040-EP060   付费意愿降低段，必须在每个 A 档钩之间穿插情绪兑现，避免"只剩压迫无回报"
EP080+        进入终局段，付费阶梯频率可降；强卡放在大反派揭露和最终对决前
```

### 2.2 S/A/B 档卡点定义

`S 档`（必拉付费的强翻面，每 10-15 集一个）：
- 不可逆翻转：身份 / 关系 / 权力 / 生死 / 真相 之一发生根本改变
- 主角处境从此完全不同
- 不能只换一句话或换一个证物；必须改变下一集的目标 / 危险 / 关系
- 典型动作：女主当众撕婚约 / 男主真实身份揭露 / 反派被公开打脸 / 主角生死悬念

`A 档`（强钩子，每 3-5 集一个）：
- 强情绪 / 强冲突 / 强反转之一
- 改变阶段任务但不一定改变全剧根本
- 典型动作：母亲手术失败 / 男主送出关键证据 / 反派开始反击

`B 档`（普通钩子，每集都要有）：
- 单集尾钩，让下一集不空降
- 改变小目标 / 小关系 / 小信息差

### 2.3 IAP 卡点的核心要求：不可逆翻转

付费卡不是给"原本的尾钩再加一句"。付费卡要让该集**结尾的关系/权力/真相状态**直接形成"不付费就看不到答案"的不可逆翻转。

判定不可逆的标准：
- 该集结束后，主角的世界观 / 处境 / 身份 / 资源 / 关系结构发生**结构性变化**
- 即使下一集允许观众想象 3 种可能走向，3 种走向都和上一集完全不同
- 反例：主角又被反派打了一次（同型重复，不算不可逆）

### 2.4 海外 episode-app（ReelShort / DramaBox 等）IAP 节奏

海外付费短剧的卡点节奏：
- 通常前 8-15 集免费，第 12-20 集首付费卡
- 每集时长 60-90 秒（与国内类似但单集结构更紧）
- 卡点形态偏 `dark romance` 高潮（强吻 / 失忆觉醒 / 三角关系崩盘 / 怀孕揭露 / 离婚撕逼）
- 付费意愿驱动：身份 / 财富 / 报复 / 浪漫高潮 是四大付费触发器

文化特化要点：
- 国内 IAP 强卡偏"公开清算 / 当众打脸 / 寒门贵子翻身"
- 海外 IAP 强卡偏"个人觉醒 / 自我重塑 / 关系炸裂 / 财富翻盘"

## 三、平台付费定制卡点（长视频平台）

适用项目：爱奇艺 / 优酷 / 腾讯视频 / Apple TV+ 等长视频平台定制项目，按集买断 / 会员独享。

卡点目标：
- 留住会员（不退订）
- 单集完播率
- 引导续订

卡点形态：
- 长头部建制（首集 / 前 3 集要立人物 / 立题材，不需要立刻强卡）
- 会员独享：在阶段大纲中标注哪些集放在会员独享段
- 中后段稳定密度：长视频用户耐心更高，但中后段坍塌率也高，每 5-8 集要有一次大事件

不适用 IAA 的"前 5 秒抓力"硬规则；但仍适用 IAA 的"追更钩"思路。

## 四、IAA vs IAP 卡点设计对照表

| 维度 | IAA | IAP |
| --- | --- | --- |
| 关键指标 | 完播率 / 留存 / CTR | 付费率 / ARPU |
| 卡点目的 | 不流失 + 立刻刷下集 | 触发付费意愿 |
| 单集内卡点 | 3 个完播抓手（5s / 25s / 50s） | 也要 3 个完播抓手 |
| 集间卡点 | 追更钩（每集都要） | B 档钩（每集）+ A 档（每 3-5 集）+ S 档（每 10-15 集） |
| 结尾设计 | 让用户行为驱动（刷下集） | 让用户付费驱动（解锁） |
| 不可逆性 | 不强求 | 强卡集必须不可逆 |
| 广告卡位 | 必须设计（中插不杀完播） | 较少考虑（除非也走免费章广告） |
| 激励视频 | 适用（IVR 解锁番外） | 不适用 |
| 阶段密度 | 平均分布 | 卡点集前后密度对比 |

## 五、跨模式通用准则

无论 IAA 还是 IAP，所有卡点都要满足：

1. `不重复同型`：相邻集尾钩不能用同一种型（不能连续两集都是身份揭露 / 不能连续两集都是背叛）
2. `具体性`：钩 / 卡的内容必须能让观众一句话复述，不允许抽象词
3. `状态改变`：每个卡点必须改变下集 / 阶段的目标 / 危险 / 关系 / 信息差至少一项
4. `行动锚点`：卡点里必须出现具体人物 / 地点 / 道具 / 时间，让下集第一场可以直接承接
5. `兑现节奏`：连续 2 集只抛卡不兑现旧债时，第 3 集必须回收一次旧债再叠新债

## 六、商业卡点和分集大纲的对接字段

仅当用户明确商业化模式时，`分集大纲.md` 才按本节追加字段。

商业化后通用字段（IAA / IAP / 平台付费定制都要）：
- `本集完播抓手`：5s / 25s / 50s 三个时间点对应的具体动作
- `本集追更钩`：尾钩内容（具体性 + 时效性 + 行动锚点 + 状态改变）
- `本集追更钩类型`：身份揭露 / 背叛 / 权力反转 / 禁忌触发 / 生死危机 / 选择困境 / 时间压力（不重复相邻集类型）

商业化后置项目不填上面三项的商业版本，但仍应在剧作层写清：
- `本集观众一句话`
- `本集情绪回报`
- `本集尾钩`
- `尾钩如何改变下一集目标 / 危险 / 关系 / 信息差`

IAA 项目额外字段：
- `本集中插广告卡位`：写在第几秒、对应正文的哪一行（如适用）
- `本集是否设激励视频`：是 / 否；若是，写清解锁的番外内容

IAP 项目额外字段：
- `本集付费触发位`：是 / 否；若是，标注 `S/A/B` 档与对应付费层级
- `本集卡点是否不可逆`：仅 S 档必填；写清不可逆体现在哪个状态

平台付费定制项目额外字段：
- `本集是否会员独享`：是 / 否；若是，写在阶段大纲对应段

## 七、卡点常见错误

- 把"反转"当卡点：不是所有反转都是好卡点；卡点必须有"具体下集需要看的理由"
- 把"密度"当抓手：信息密度高不等于完播抓手到位；抓手必须是可见动作或可见冲击
- 把"暗示"当钩子：抽象暗示（"故事即将进入新阶段"）不是钩子；钩子必须可一句复述
- 把"未解之谜"当追更：纯粹未解之谜让观众等不一定刷下集；必须叠加"看了立刻爽"或"看了立刻揪心"的情绪
- 把"中插随便插"当广告卡位：不在情绪余震切广告会直接杀完播；广告卡位是写作设计的一部分，不是后期处理
- 海外项目套用国内 IAP 卡点：海外文化爽点不同，公开打脸 / 寒门翻身在海外不是默认强卡
