# 国内选题与设定词库

这份词库用于 `/立项` 和 `/策划`。

用途：
- 帮助快速组织选题
- 帮助把题材、设定、风格拆成可组合的创作维度
- 帮助在策划阶段把“好看”落成具体故事引擎

使用原则：
- 这是词库，不是死表单
- 可以直接选用词库里的词，也可以在此基础上自定义
- 不要为了显得丰富而堆满维度
- 默认先锁 `目标受众 1 个 + 题材类型 1-2 个 + 核心设定 2-4 个 + 风格元素 2-4 个`
- 选定后要回答：这些组合为什么适合目标市场、为什么能持续连载、为什么能写成戏

## 一、目标受众

- 男频（主爽点偏升级、逆袭、权力、征服）
- 女频（主爽点偏关系、情绪、身份反杀、公开清算）
- 泛受众（主爽点偏事件、悬疑、灾变、规则）

使用提醒：
- `男频 / 女频` 先决定主爽点、关系结构和回报方式
- `泛受众` 适合悬疑、家庭、灾变、规则世界等强事件驱动方向

### 价值观爽点池（国内主流）

- `寒门贵子`：低起点逆袭，强调出身和反差
- `扮猪吃老虎`：藏锋不露，关键时刻翻面
- `装弱反杀`：被压到底再一击炸开
- `以德报德 / 以直报怨`：亲人朋友护到底，敌人按规则打回
- `家国情怀`：个人选择和大义同向
- `公开清算`：在公开场让对手社死、当众撕下面具
- `投射打工人`：被领导 PUA / 被甲方羞辱 / 被同事抢功后翻盘
- `豪门下嫁 + 反向追妻`：女主先冷男主再追

价值观反面（默认避免，需要时显式标注）：
- `绿茶 / 白莲花`：装单纯实则心机；反面角色用，不要给主角
- `PUA`：精神控制；只能写给反派或写给主角早期被害然后反杀
- `捞女 / 凤凰男`：地域 + 性别污名化标签，谨慎使用
- `躺平`：真躺平没爽感；剧本里的"躺平"通常是假装放弃，最后再翻
- `富即正义 / 拜金`：直接拜金违反平台价值观，要包装成 `事业 + 自尊`

### 地域分层与城市绑定

不同地域有天然剧情纹理，选题时优先匹配：

- 二三线 / 县城：人情社会 / 国企家庭 / 体制内 / 房子结婚 / 回家相亲 / 小镇做题家
- 东北：下岗 / 重工业 / 江湖气 / 喜剧底色 / 大碴子味
- 广东 / 福建：宗族 / 祠堂 / 港台情结 / 出海 / 茶饭文化
- 川渝：麻将 / 茶馆 / 火锅 / 江湖底色 / 麻辣感
- 江浙沪：吴侬软语 / 家族企业 / 外贸 / 学区房军备竞赛

提醒：
- 选定地域后，对白、场景、配角都要顺着这条纹理走；不要一边主角说北方话一边场景写成上海弄堂
- AI 漫剧不强制视觉地域感，但场景符号、招牌、菜系、街头细节要对得上

## 五、组合规则

优先考虑：
- `题材类型` 决定世界和主冲突
- `核心设定` 决定主角引擎和爽点来源
- `风格元素` 决定情绪和传播气质

避免问题：
- 不要同时堆过多题材类型，默认 `1-2` 个足够
- 不要同时堆过多核心设定，默认 `2-4` 个足够
- 不要让风格互相打架，例如一边走惊悚诡秘，一边又做高频沙雕玩梗
- 不要把 `流派 / 身份 / 金手指 / 情节` 全部堆满，最终要回到一个清晰的主引擎

## 六、策划时必须回答

从词库中选完组合后，必须进一步回答：
- 这个组合为什么适合目标受众
- 这个组合的核心爽点或情绪点是什么
- 观众为什么会一直看下去
- 这个组合最怕写成什么老套样子
- 中段如何变奏，不变成同一招重复
- 这个组合对角色、场景、道具提出了什么资产要求
