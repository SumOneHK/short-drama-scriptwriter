# 规格建议

本文件用于在不同发行平台、目标市场和故事规模下，为 AI 竖屏剧 / 漫剧项目选择更合理的集数与时长规格。

## 〇、长线项目阈值

固定口径：
- `总集数 >= 30`：长线项目，`阶段大纲.md` 必出，正文阶段每批写作 / 复盘 / 重复风险检查都按长线项目口径执行
- `总集数 20-29`：短中线项目；中段戏剧形态明显变化、双时间线、多主线或强设定时，仍建议按长线项目处理
- `总集数 < 20`：短线项目，默认不出阶段大纲

## 一、先看平台、目标市场和故事规模

优先按发行平台和故事规模选规格，而不是只按题材或商业化模式选。

商业化模式默认后置；除非用户明确要求 IAA / IAP / 平台付费定制，不用商业化模式倒推规格。规格首先服务：首集建制、类型承诺、人物欲望、阶段回报和竖屏观看节奏。

### 抖音国内端原生 / 信息流分发

适合：
- 强钩子、强反转、强互动
- 需要快速起量
- 依赖短视频原生流量

建议：
- 总集数：`30-60` 集
- 常规单集：`60-90` 秒
- 首集：`90-150` 秒

### 国内免费短剧 APP / 长线连载

适合：
- 更强调留存与总观看时长
- 可以接受更长的连载
- 适合家庭、情感、成长、男频升级等长线追更题材

建议：
- 总集数：`60-90` 集
- 常规单集：`60-90` 秒
- 首集：`100-180` 秒

### 国内精品短剧 / 平台项目 / 更高叙事规格

适合：
- 明确精品路线
- 平台定制、品牌合作、重点项目
- 允许更完整叙事承载

建议：
- 总集数：`25-40` 集 或 `30-60` 集
- 常规单集：`90-120` 秒 或 `2-5` 分钟
- 首集：常规集的 `1.3-1.8` 倍

### TikTok 出海 / 美国英语首版

适合：
- TikTok 竖屏观看和社媒传播
- Romance / dark romance / supernatural / romantasy / Isekai-lite / thriller 等类型承诺清楚的题材
- 更依赖首屏可懂、前几集类型兑现和可切片高光

建议：
- 总集数：`50-80` 集
- 常规单集：`45-75` 秒
- 首集：`75-120` 秒

### 海外独立短剧 App / 付费系列

适合：
- 用户明确要求 paywall 或 episode-app 付费设计
- 高频 cliffhanger
- 用户连续解锁

建议：
- 总集数：`60-100` 集
- 常规单集：`45-75` 秒 或 `60-90` 秒
- 首集：`75-120` 秒

## 二、按题材微调

### 都市甜宠 / 强关系拉扯

- 用中等集数更稳：`30-60` 集
- 单集不宜太长，否则甜感会被稀释

### 男频升级 / 复仇 / 家族 / 系统流

- 更适合长线：`60-90` 集
- 因为天然支持“任务-升级-对手升级”的循环

### 悬疑 / 惊悚 / 校园惊险

- 集数不宜无限拉长，建议 `30-60` 集
- 否则谜底会被拖垮

### 世界观型奇幻 / 玄幻 / AI 漫剧

- 如果卖点在设定和奇观，允许 `60-90` 集
- 但必须设计清楚能力系统、情绪回报和阶段升级，否则会变成设定堆叠

## 三、首集为什么常常更长

首集加长通常不是偶然，而是为了同时完成下面几件事：
- 让用户在第一集内看懂主角处境
- 让用户看到核心卖点，不只是听介绍
- 更完整地交付一次“情绪兑现”
- 给后续切片素材和高光传播更大空间
- 为前 `5-10` 集的留存和类型承诺打底

如果项目后续明确商业化，首集也可承担转化漏斗；但在剧本阶段，首集加长首先服务建制和情绪兑现。

## 四、是否应该把“首集加长”写进 skill

建议写进，但不要作为死规则。

推荐规则：
- 默认允许首集比常规集长 `1.5-2.0` 倍
- 只有当首集承担了更大的建制任务时才使用
- 若首集只是慢、不是强，就不要加长

## 五、默认值

在用户没有指定规格时，建议默认：
- 抖音国内：`40` 集左右，常规集 `70-90` 秒，首集 `110-140` 秒
- TikTok 出海 / 美国英语首版：`60` 集左右，常规集 `50-70` 秒，首集 `80-110` 秒
- AI 竖屏剧 / 漫剧：在对应平台规格基础上，优先锁清角色识别锚点、视觉符号、规则后果和阶段回报

## 六、AI 制作风格建议

不同 AI 制作风格会影响角色识别、画面语言和视觉锚点。这里不是为了限制题材，而是帮助在立项和策划阶段把风格与故事承诺配上。详见 [23-AI漫剧制作约束.md](23-AI漫剧制作约束.md)。

| 子风格 | 角色识别重点 | 画面表达优势 | 适合剧本类型 | 注意 |
| --- | --- | --- | --- | --- |
| 写实 / AI 真人 | 面孔、发型、服装锚点 | 现代爱情 / 悬疑 / 职场真实感 | 都市 / 现代爱情 / 悬疑 | 不要只靠微表情承载关键反转 |
| 3D（CG 风） | 轮廓、服装、道具 | 大场景、奇幻空间、动作感 | 玄幻 / 仙侠 / 奇幻 / 规则世界 | 情绪戏要用动作和道具补足 |
| 国漫 | 发型、服饰、法器 | 东方奇幻、都市奇幻、强视觉符号 | 玄幻 / 都市奇幻 / 校园 | 目标市场出海时要注意类型语法重写 |
| 日漫 | 发型、眼睛、制服/服饰 | 学院、异世界、轻奇幻、表情夸张 | 校园 / Isekai / villainess / 治愈 | 台词不要过度二次元化，除非目标受众明确 |
| 欧美漫画 / romantasy | 服装剪影、首饰、色彩系统 | 哥特、狼人、吸血鬼、魔法学院 | dark romance / romantasy / gothic curse | 规则和关系要一眼可懂 |
| 赛博 / 暗黑 | 光色、机械/纹身/界面 | 末世、科幻、规则世界、黑暗悬疑 | survival / sci-fi / noir romance | 不要让视觉噪音淹没人物欲望 |
| 萌系 / Q 版 | 夸张轮廓、固定道具 | 萌宠、童趣、轻喜、轻奇幻 | 萌宠 / 童趣 / 日常 | 不适合强虐或强成人情绪 |

总集数对应资产规划（粗估，不作为选题否决）：
- 30-40 集：先锁主角、核心关系位、核心对手和 3-5 个高复用视觉场景
- 60-80 集：先锁阶段场景、阶段造型、阶段规则和阶段回报
- 100+ 集：必须有阶段大纲明确划分阶段角色 / 阶段场景 / 阶段视觉系统

具体 AI 制作口径见 [23-AI漫剧制作约束.md](23-AI漫剧制作约束.md)。
