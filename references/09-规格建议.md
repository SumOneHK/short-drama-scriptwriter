# 规格建议

本文件用于在不同平台、不同发行模式下，为短剧项目选择更合理的集数与时长规格。

## 〇、长线项目阈值

固定口径：
- `总集数 >= 30`：长线项目，`阶段大纲.md` 必出，正文阶段每批写作 / 复盘 / 重复风险检查都按长线项目口径执行
- `总集数 20-29`：短中线项目；中段戏剧形态明显变化、双时间线、多主线或强设定时，仍建议按长线项目处理
- `总集数 < 20`：短线项目，默认不出阶段大纲

## 一、先看平台和模式

优先按发行模式选规格，而不是只按题材选。

每种发行模式都隐含一种变现模式（IAA / IAP / 平台付费定制）。卡点设计按变现模式不同走完全不同的路径，详见 [24-卡点设计.md](24-卡点设计.md)；规格只是承载卡点的容器，先锁变现模式再选规格。

### 抖音国内端原生 / 信息流分发

付费分布：偏免费/广告变现，IAA 为主；少量项目挂小程序付费卡点。

适合：
- 强钩子、强反转、强互动
- 需要快速起量
- 依赖短视频原生流量

建议：
- 总集数：`30-60` 集
- 常规单集：`60-90` 秒
- 首集：`90-150` 秒

### 国内免费短剧 APP / IAA 为主

付费分布：免费观看，广告变现；中段无强卡点，但每集尾钩拉力直接影响留存。

适合：
- 更强调留存与总观看时长
- 可以接受更长的连载
- 适合家庭、情感、成长、男频升级等长线追更题材

建议：
- 总集数：`60-90` 集
- 常规单集：`60-90` 秒
- 首集：`100-180` 秒

### 国内精品短剧 / 平台项目 / 更高制作规格

付费分布：平台定制 / 品牌合作 / 会员独享；通常按 `平台付费定制` 处理，不走小程序付费卡点。

适合：
- 明确精品路线
- 平台定制、品牌合作、重点项目
- 允许更完整叙事承载

建议：
- 总集数：`25-40` 集 或 `30-60` 集
- 常规单集：`90-120` 秒 或 `2-5` 分钟
- 首集：常规集的 `1.3-1.8` 倍

### TikTok 欧美首版

付费分布：免费观看 + 广告 / 切片导流到独立 App；本身不挂付费卡点，但首屏 `0-15s` Hook 的强度直接决定转化。

适合：
- 社媒冷启动与广告切片投放
- Romance / dark romance / supernatural / thriller 等高概念题材
- 更依赖首屏钩子和前几集转化

建议：
- 总集数：`50-80` 集
- 常规单集：`45-75` 秒
- 首集：`75-120` 秒

### 海外独立短剧 App / 付费系列

付费分布：典型 IAP 付费阶梯——免费章 `EP001-EP010` 拉钩；首付费卡 `EP010-EP015` 必须放 `S` 档强翻面；`EP015` 之后按解锁阶梯每 `3-5` 集一次 `A` 档以上钩子。

适合：
- 明确的 paywall 设计
- 高频 cliffhanger
- 用户连续解锁

建议：
- 总集数：`60-100` 集
- 常规单集：`45-75` 秒 或 `60-90` 秒
- 首集：`75-120` 秒

## 二、按题材微调

### 都市甜宠 / 强关系拉扯

- 用中等集数更稳：`30-60` 集
- 单集不宜太长，否则甜感会被稀释

### 男频升级 / 复仇 / 家族 / 系统流

- 更适合长线：`60-90` 集
- 因为天然支持“任务-升级-对手升级”的循环

### 悬疑 / 惊悚 / 校园惊险

- 集数不宜无限拉长，建议 `30-60` 集
- 否则谜底会被拖垮

### 世界观型奇幻 / 玄幻 / AI 漫剧

- 如果卖点在设定和奇观，允许 `60-90` 集
- 但必须设计清楚能力系统和资源复用，否则制作成本会失控

## 三、首集为什么常常更长

首集加长通常不是偶然，而是为了同时完成下面几件事：
- 让用户在第一集内看懂主角处境
- 让用户看到核心卖点，不只是听介绍
- 更完整地交付一次“情绪兑现”
- 给后续投流素材和切片素材更大空间
- 为前 `5-10` 集的免费转化或留存打底

对付费模式（IAP）尤其重要，因为首批免费集既是内容，也是付费转化漏斗。对 IAA 模式同样重要，首集是否抓人直接决定后续完播曲线。

## 四、是否应该把“首集加长”写进 skill

建议写进，但不要作为死规则。

推荐规则：
- 默认允许首集比常规集长 `1.5-2.0` 倍
- 只有当首集承担了更大的建制任务时才使用
- 若首集只是慢、不是强，就不要加长

## 五、默认值

在用户没有指定规格时，建议默认：
- 抖音国内：`40` 集左右，常规集 `70-90` 秒，首集 `110-140` 秒
- TikTok 欧美：`60` 集左右，常规集 `50-70` 秒，首集 `80-110` 秒
- AI 漫剧：在对应平台规格基础上，优先控制人物数、常驻场景数、角色造型数和特效系统数

## 六、AI 漫剧子风格成本表

不同 AI 漫剧子风格的制作成本差异极大；选风格时一并锁制作约束。详见 [23-AI漫剧制作约束.md](23-AI漫剧制作约束.md)。

| 子风格 | 角色一致性难度 | 单场镜头预算 | 适合剧本类型 | 不建议 |
| --- | --- | --- | --- | --- |
| 写实 / AI 真人 | 极高（最难） | 3-4 镜头 | 都市 / 现代爱情 / 悬疑 | 高动作 / 群像 / 跨时空 |
| 3D（CG 风） | 中 | 4-6 镜头 | 玄幻 / 仙侠 / 奇幻 | 强微表情戏 |
| 国漫 | 中 | 4-5 镜头 | 玄幻 / 都市奇幻 / 校园 | 极写实场景 |
| 日漫 | 低-中 | 4-6 镜头 | 校园 / 治愈 / 萌系 | 极写实 / 强情绪写实 |
| 赛博 / 暗黑 | 中 | 3-5 镜头 | 末世 / 科幻 / 黑暗悬疑 | 阳光甜宠 |
| 萌系 / Q 版 | 低 | 5-7 镜头 | 萌宠 / 童趣 / 日常 | 强情绪悲剧 |

总集数对应资产预算（粗估）：
- 30-40 集：A 级常驻场景 ≤ 4 个，主角 + 核心关系位 + 核心对手共 3-4 套造型
- 60-80 集：A 级常驻 ≤ 6 个，常驻角色 5-7 个，每人 3-5 套造型
- 100+ 集：A 级常驻 ≤ 8 个，但必须有阶段大纲明确划分阶段角色 / 阶段场景

具体制作成本（廉价 vs 昂贵的写法转化）见 [23-AI漫剧制作约束.md](23-AI漫剧制作约束.md) 第一节。
