# 立项层细则

## 目标

- 明确发行平台、目标市场、AI 风格、题材、受众、规格和必要边界
- 所有项目都先完成最小 `市场判断`
- 在立项阶段先锁 `内容形态 / 对白语言要求 / 禁用元素 / 改编来源 / AI 制作口径`
- 如果用户题材还没定，先做选题决策，不直接写剧情
- 如果输入是小说或长文本，先整理纯中文故事理解，不直接硬转剧本
- 调研、市场快照和待选题材池只能服务决策；用户没有明确确认最终方向和必要立项信息前，不得初始化项目目录，不得落任何立项文件
- 高概念不是默认门槛，现实题材也不是默认优先。立项先判断选题策略：`主流稳态 / 类型微创新 / AI-native 视觉型 / 高概念创新`，再组织候选池
- 平台和市场分开：国内平台写 `抖音`；出海平台写 `TikTok`。目标市场另写 `国内 / 美国英语首版 / 泛欧美 / 东南亚 / 拉美 / 指定国家或地区`
- 剧本阶段默认商业化后置；除非用户明确要求，不用 IAA / IAP / 付费卡点来倒推选题

## 流程

题材未锁项目：
1. 对话内市场判断
2. 对话内候选题材池（默认覆盖不同选题策略，不得全部压成高概念）
3. 用户明确确认最终方向、发行平台、目标市场、受众、规格、AI 风格等必要立项信息
4. 初始化项目目录并锁题落盘

题材已锁原创项目：
1. 轻量市场判断（内嵌）
2. 确认必要立项信息
3. 初始化项目目录并生成项目简报 / 锁题摘要

题材已锁改编项目：
1. 对话内故事理解
2. 轻量市场判断（内嵌）
3. 确认必要立项信息
4. 初始化项目目录并生成项目简报 / 锁题摘要

## 输出

- `项目简报.md`（仅用户明确确认方向并进入 `T02` 后）
- `市场快照.md`（确认后按需归档；未确认阶段不落盘）
- `待选题材池.md`（确认后按需归档；未确认阶段不落盘）
- `锁题摘要.md`（仅用户明确确认方向并进入 `T02` 后）
- `故事理解.md`（仅改编项目确认后按需归档；未确认阶段不落盘）

产物稳定口径：
- `市场判断` 是所有项目的必做动作
- 进入 `T02` 后，`项目简报.md` 记录用户原始需求、确认信息、约束和轻量市场判断；`锁题摘要.md` 记录最终创作决策、核心卖点、连载引擎和不可偏离边界
- 两份文件共享平台、市场、规格、确认记录等硬字段，但不重复展开同一段市场分析、题材论证或卖点推导
- `市场快照.md` 和 `待选题材池.md` 不是未确认阶段的落盘文件；它们首先是对话组织材料，只在用户确认后需要归档时创建
- 如果题材、市场和项目边界一开始就被用户明确锁定，可以直接产出 `项目简报.md + 锁题摘要.md`
- 如果用户只是在讨论、比较、要求继续调研或说“再看看”，视为未锁题；只在对话中继续汇总，不更新任何项目文件，不生成 `项目状态.json / 质检检查点.md / 锁题摘要.md`
- 任何“为了让目录看起来完整”而补建的立项文件，都视为无效文档膨胀

## 选题策略分流

立项候选池先判断项目更适合哪类策略，不把”新奇设定”误当成唯一卖点，也不因 AI 制作误判为只能做现实场景题材：

- `主流稳态`：复仇、逆袭、身份反转、豪门、玄幻升级、关系拉扯等成熟引擎；重点是爽点清楚、回报稳定
- `类型微创新`：在成熟题材里换场域、压迫机构、人物关系或视觉符号；重点是降低同质化，同时不增加理解门槛
- `AI-native 视觉型`：选择 AI 竖屏剧能放大的题材，如 romantasy / Isekai / villainess / werewolf / vampire / witch / academy / gothic curse / apocalypse / rules world；重点是视觉记忆点、规则可读性和情绪引擎，而不是压低场面规模
- `高概念创新`：规则反转、因果反转、时间线、身份系统、世界机制等；重点是一眼能懂，不能靠长解释撑卖点

### 选题策略决策树（按以下四问选档，避免默认推高概念或默认现实化）

写候选池前，对每个候选方向先按这四问打分：

1. **规则复杂度**（这个题材一句话能讲清吗？）
   - 普通观众一句话能懂 → 加分给 `主流稳态` / `类型微创新`
   - 必须先解释 1-2 个新规则才懂 → 加分给 `类型微创新` / `高概念创新`
   - 必须先解释 ≥ 3 个新规则、否则连前两集都看不懂 → 高概念创新；但必须能给出”如何在 EP001 前 60 秒看懂规则后果”的具体设计，否则建议改成主流稳态

2. **爽点清晰度**（每集观众拿到什么？）
   - 每集有具体可见的赢 / 输 / 翻面（关系 / 身份 / 资源 / 公开打脸）→ 加分给 `主流稳态` / `类型微创新`
   - 爽点依赖规则解锁、机制理解或长链反转 → 加分给 `高概念创新`；但必须能在前 5 集兑现 ≥ 1 次具体回报，否则风险高
   - 爽点说不清或停留在”主角变强 / 主角逆袭” → 任何策略都会塌，回改卖点

3. **AI 视觉红利**（这个题材是否比真人短剧更适合 AI 制作？）
   - 有一眼可记的视觉符号、类型氛围或世界图像（如红月婚礼 / 诅咒戒指 / 魔法学院 / 狼族血誓 / 末日规则屏）→ 加分给 `AI-native 视觉型`
   - 视觉主要是办公室、客厅、会议室且缺少独特符号 → 不降级，但要补具体视觉锚点
   - 大场面、奇幻、跨时空、群像不是立项否决项；只要求角色识别、规则后果和关键视觉锚点可复用

4. **平台观看语法**（TikTok / 抖音首屏是否可懂？）
   - 第一场就能看见危机、欲望、规则后果或关系撕裂 → 加分
   - 需要靠长旁白解释前史 / 世界观 / 商业模式 → 回改入口
   - 能切出 3-5 个可独立传播的高光场面 → 加分

判定矩阵：
- 全部都是”加分给主流稳态”的方向 → 推 `主流稳态`，先看能不能在场域 / 压迫机构 / 视觉符号上做 `类型微创新`
- 视觉红利强、规则可懂、类型承诺清楚 → 推 `AI-native 视觉型`
- 同时命中”规则复杂”+ “爽点依赖规则”+ “首屏能看见规则后果” → 才推 `高概念创新`；并且必须给出”前 60 秒规则后果落地方案”
- 拿不准时默认推 `主流稳态` 或 `类型微创新`，不默认推 `高概念`

题材未锁时，对话内候选池默认至少给出 `2-4` 个不同策略的方向；如果只给 `1` 个方向，必须说明这是用户已指定的窄范围，不是系统自动锁题。

`高概念创新` 的最小条件（不满足则不推）：
- 一句话能让普通观众讲清”这部剧的规则是什么”
- EP001 第一场前 4 行能让观众看见规则后果（不是听人解释规则）
- EP001 至少有一次”被规则压迫”的具体动作场面
- 前 5 集至少兑现 1 次具体回报（不是只解锁更多规则）

## 合规与题材敏感线

立项阶段除了"能不能连载、能不能拍"，还要轻量预判"会不会被打回"。这一层不替代正式法务/平台合规审查，但可避免明显送命题材进入策划。

最小动作：
- 在 `项目简报.md` 的 `合规风险预判` 字段写清当前题材最可能踩到的 `1-3` 条平台/监管红线
- 命中敏感题材时，`题材敏感点` 字段必须显式标注

平台与监管常见敏感题材（举例，不穷尽）：
- 涉未成年：校园暴力、未成年人恋爱、未成年人涉色情
- 涉公职：警察、检察官、法官、军人形象不严肃化或反转化
- 涉医：虚假诊疗效果、医患极端冲突、药物乱用
- 涉法：私刑、教唆犯罪、把违法行为正当化
- 涉民族宗教：少数民族文化符号、宗教仪式、特定宗教人物
- 历史人物 / 真实事件改编：未授权使用真实姓名、明显歪曲历史
- 价值观红线：拜金、炫富、复仇至上、家暴美化、PUA 关系美化
- 海外项目额外：宗教禁忌、种族刻板印象、性别议题极化、未成年人涉性

国内项目额外动作：
- 若涉及上述敏感线，在 `锁题摘要.md` 的 `合规与边界` 段写清规避策略
- 长线 / 平台定制项目建议提示用户走平台立项备案绿色通道

海外项目额外动作：
- 美国英语市场项目要按 TikTok / Apple App Store 内容政策预判 `性 / 暴力 / 自残 / 仇恨言论` 红线
- 涉枪、涉毒、涉自残必须显式标注降级策略

合规预判与禁用元素的区别：
- `禁用元素`：项目自己主动不做的内容（创作意愿）
- `合规风险预判`：项目即使想做也不能做或要降级处理的内容（外部约束）

## 过关条件

- 题材已经被用户确认，而不是系统自动默认
- `项目简报.md / 锁题摘要.md` 已写清用户确认依据；如果没有用户明确确认，立项质检必须判为 `需修改`
- 目标受众、题材类型、核心设定、风格元素已经形成可执行的基础组合
- `内容形态 / 发行平台 / 目标市场 / 对白语言要求 / 禁用元素 / AI 风格` 已写清，不把项目边界问题留到 `/策划` 或 `/复盘`
- 国内项目已按国内词库组织；美国英语市场项目已按美国英语市场词库组织
- 规格与平台匹配
- 长线项目没有遗漏长线结构要求
- 已形成最小市场判断，至少说清当前题材空间、同质化风险和差异化抓手
- 已完成最小合规预判：`项目简报.md / 锁题摘要.md` 至少说清最可能踩到的 `1-3` 条平台/监管红线及规避策略；命中敏感题材时已显式标注
- 若用户明确要求商业化，已锁定商业化模式与对应卡点策略骨架；若用户未要求，明确记录为 `商业化后置`
- `锁题摘要.md` 已说明为什么这个组合能连载、为什么适合 AI 竖屏呈现、前三集抓手和最大风险
